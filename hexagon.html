Hexagon CCI

My Data Analytics internship experience from Sept 2022.

Experience

As a Software Engineer Intern - Data Analystics at Hexagon, I led the development of a data automation solution that significantly improved data integrity and process efficiency. I built and deployed a Python-based data cleaning pipeline, scheduled weekly via Windows Scheduler on a virtual machine, saving over 10 hours of manual effort. I executed complex SQL queries on Microsoft SQL Server to retrieve and analyze key datasets used in over 20 business reports, ensuring timely and accurate insights for stakeholders. To enhance accessibility and stakeholder engagement, I designed and launched a web application using Django that enabled 50+ users to view cleaned data through a user-friendly interface. I also collaborated with the UX team to iterate on UI improvements based on usability feedback. This end-to-end ownership—from automation to UI deployment—sharpened my ability to manage technical delivery, coordinate across teams, and align data products with business needs.
