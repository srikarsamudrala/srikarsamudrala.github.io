My Data Analytics internship experience from Sept 2022.

Experience

As a Data Analyst Intern at Hexagon, I developed automation solutions to enhance data accuracy and streamline processes. Additionally, I conducted SQL queries to retrieve and analyze relevant data for reporting. To enhance accessibility, I built a web application from scratch using Django, providing a user-friendly interface for viewing cleaned data. Collaborating with the UX team, I contributed to refining the WebApp’s UI for a better user experience.
